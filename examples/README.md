# Examples

Real blueprints the skill produced on randomly generated briefs, kept as a public showcase and a
regression set. Each was scored by the multi-role council (see [`../docs/VALIDATION.md`](../docs/VALIDATION.md)).

| Example | Track | Council score |
|---|---|---|
| [offshore-wind-decommissioning](offshore-wind-decommissioning/) | A · greenfield (litigation-exposed) | **9.10 / 10** |

Each folder holds the `brief.md` (the input, generated blind) and the `blueprint.md` (the skill's
output). Contributions of new validated examples — different tracks, different domains, with their
council scores — are welcome.
