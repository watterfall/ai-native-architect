# The T1 Canvas · 判断 × 上下文 画布

> One page. Fill the blanks. It forces the only two questions that matter in an AI-Native design:
> **where does judgment sit, and how does context flow?** No AI needed; ~10 minutes with a team.
>
> 一页纸,填空即可。它逼你回答 AI-Native 设计里唯一重要的两问:**判断在哪里、上下文怎么流动?**
> 不用 AI,团队一起约 10 分钟。

> **T1 · the theorem:** organization = **distribution of judgment** × **flow of context**. Scale is a free
> variable; management is the engineering of these two structures. · **组织 = 判断的分布 × 上下文的流动**。
> 规模是自由变量;管理,就是这两张结构的工程学。

## 1 · Distribution of judgment · 判断的分布
List the **few** irreducible human nodes. Everything else defaults to an agent. Fewer is better; justify

> Gut check: if a human is on *every* edge, you have replicated the old org. Cut until only the
> irreducible judgments remain. · 自检:若每条边上都有人,你只是复制了旧组织。砍到只剩不可替代的判断为止。

## 2 · Flow of context · 上下文的流动
The compounding context system: what gets **written**, **where it lives**, **who/what reads it back**.
Without it your agents are generic; with it they become uniquely yours. · 复利的上下文系统:写**什么**、
存**哪里**、下游**谁/什么**读回。没有它 agent 是通用的;有了它,它们才独属于你。

| Written · 写入 | Lives in · 存放 | Read back by · 读回方 |
|---|---|---|
| (e.g. decisions + rationale) | (e.g. a decision log) | (the next run; the successor) |
| | | |
| | | |

> Gut check: does year two get structurally smarter than year one? If knowledge still lives in heads and
> decays, it is not native. · 自检:第二年是否结构上比第一年更聪明?若知识仍在人脑里衰减,就不是原生。

## 3 · The four-layer substrate · 四层底座
Tick what you have; name what you defer (and when you'd add it). · 有的打勾,延后的写明(以及何时添)。

- [ ] **Model** · 模型层 — ≥1 frontier API + open-weight for sovereign work, behind a swap abstraction.
  *Single-vendor = API-dependent, not native.* ____________________
- [ ] **Agent** · Agent 层 — orchestration + runtime + tool/data integrations. ____________________
- [ ] **Context** · 上下文层 — vector store / knowledge graph / decision logs + a freshness practice.
  *This is what makes the agents yours.* ____________________
- [ ] **Observability** · 可观测层 — logs, traces, evals, alerts, a human review queue.
  *Without it you scale failure.* ____________________
