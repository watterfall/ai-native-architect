# Lite tools · 轻量工具(从这里开始)

**Start here if you don't use Claude Code.** These are no-install, low-barrier tools, simplest first. The
[skill](../skills/ai-native-architect/) is the full, rigorous, council-validated version; these are the
fast on-ramps. All share one vocabulary with the methodology (T1, the 16 bottlenecks, the four-layer
substrate, the kernel).

**不用 Claude Code 就从这里开始。** 这些是无需安装、低门槛的工具,由易到难。[skill](../skills/ai-native-architect/)
是完整、严谨、经评审验证的版本;这些是快速入口。它们与方法论共享同一套词汇。

| Tool · 工具 | Type · 类型 | Barrier · 门槛 | What you get · 产出 |
|---|---|---|---|
| [self-test.md](self-test.md) | decision card · 自测卡 | pen + paper, ~2 min · 纸笔 2 分钟 | Are you AI-Native or AI-enabled? Which track? An honest verdict. |
| [bottleneck-scorecard.md](bottleneck-scorecard.md) | diagnostic · 诊断表 | pen + paper · 纸笔 | A 0–16 score and the band you're in. |
| [t1-canvas.md](t1-canvas.md) | planning canvas · 画布 | fill-in, ~10 min · 填空 10 分钟 | Your judgment distribution × context flow, on one page. |
| [portable-prompt.md](portable-prompt.md) | prompt · 提示词 | paste into any chatbot · 粘进任意聊天框 | An AI-assisted lite blueprint, no install. |

### A suggested path · 建议路径
1. **self-test** — am I even the target? Which track? · 我是不是目标群体?哪条轨?
2. **t1-canvas** (greenfield) or **bottleneck-scorecard** (existing org) — find the shape. · 找出形状。
3. **portable-prompt** — get an AI-assisted draft in any chatbot. · 在任意聊天框拿到 AI 起草。
4. **the skill** — when you want the rigorous, full blueprint. · 想要严谨完整蓝图时,上 skill。
