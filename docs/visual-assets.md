# Visual Assets

These images are explanatory visual anchors for the site. They support module comprehension and orientation; they are not evidence, source material, or proof of any claim.

## Common Style Constraints

- Restrained editorial atlas style.
- No generic AI sci-fi treatment.
- No embedded text inside the image itself.
- Clear 4:3 composition suitable for cards and responsive image slots.
- Project-bound generated imagery saved locally in the repository.
- Final delivery format: WebP, 1200x900, quality 90.
